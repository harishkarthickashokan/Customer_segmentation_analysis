# Customer Segmentation using K-Means Clustering

This project performs customer segmentation using K-Means clustering based on annual income and spending score. The goal is to identify meaningful customer groups for targeted marketing and business strategies.

## Features
- Data preprocessing & feature selection
- K-Means clustering
- Scatter plot visualization
- Business insights based on cluster behavior

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-Learn
- Matplotlib
- Streamlit (for deployment)

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
